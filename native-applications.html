--- layout: default title: Native Applications description: OnChrome needs a native application to connect to Chrome. Download it here. ---

The only way for an extension to connect to an application on your machine is via Native Messaging. OnChrome needs to do exactly that to open Chrome.

You can view the source code and compile them yourself if you prefer. I'm providing the binaries because I assume it's going to be the easiest route for 99% of the users.

{% header h2 | Installation %}

Install the extension from the Firefox add-on store.

{% header h3 | macOS %}

If you use homebrew, you can run brew cask install g3rv4/casks/onchrome

If you don't, download and install OnChrome.pkg.

{% header h3 | Windows %}

The application is signed, but Windows shows a warning because I don't have enough reputation.

  1. Download and execute OnChrome.msi.
  2. You will see a blue screen saying that Windows protected your PC. Click on "More info".
  3. Verify that it shows my name.
  4. Click on "Run anyway".
